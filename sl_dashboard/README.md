# SL-Dashboard

Dieser Ordner ist der Startpunkt fuer die Entwicklung eines eigenen
SL-Dashboards innerhalb der ArdanosWiki.

Ziel:

- eine leitungsorientierte Oberflaeche statt eines reinen Wiki-Startbilds
- schneller Zugriff auf Szenen, NSCs, Orte, Encounter und Sitzungsnotizen
- klare Trennung zwischen Nachschlagewerk und aktivem Leiten
- Szenen koennen als Obsidian-taugliche Markdown-Dateien mit Frontmatter gepflegt werden

Empfohlener Start fuer die Entwicklung:

- Konzepte und Anforderungen zuerst in `notizen/` sammeln
- danach Datenmodell fuer Sitzungen, Szenen und Pins festlegen
- anschliessend erste Streamlit-Komponenten fuer das Cockpit bauen

Naechste sinnvolle Bausteine:

- Sitzungsmodell definieren
- Pins und Schnellzugriffe planen
- aktive Szene als zentrale Dashboard-Karte entwerfen
- Encounter-Rechner und NSC-Ansicht integrieren
